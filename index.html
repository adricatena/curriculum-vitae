Proyecto Final de Ingenieria Electronica
Huerta Inteligente con tecnologias IoT

Estoy finalizando mi Tesis para recibirme como Ingeniero en Electrónica de la UTN. El mismo es un proyecto casi en su totalidad de desarrollo, basado en diferentes tecnologías IoT.
Del lado del cliente se va a desarrollar una app para Android (la cual mi compañero de proyecto está a cargo), que está conectada a través de los distintos servicios que provee Firebase, como Firestore y Cloud Functions, a una Máquina Virtual hospedada dentro del ambiente de Google Cloud Platform.
En este servidor están corriendo como servicios tanto el broker MQTT como un script de "traducción" MQTT-Base de Datos. Este broker se encarga de las comunicaciones vía MQTT con un microcontrolador con capacidad de conexión a internet (en este caso es un ESP32). Dicho microcontrolador es programado en C++ para recibir y transmitir información proveniente de los distintos sensores distribuidos en el sistema como también activar/desactivar diferentes actuadores de riego.
